# تنزيل القرآن الكريم

<div dir="rtl">

تطبيق ويب تفاعلي لعرض آيات القرآن الكريم مرتبة حسب التنزيلة (ترتيب النزول).

## المميزات

- **عرض حسب التنزيلة**: عرض آيات القرآن مرتبة حسب ترتيب النزول
- **عرض حسب السورة**: تصفح القرآن حسب السور التقليدية
- **بحث تفاعلي**: تصفح الآيات والمجموعات بسهولة
- **واجهة عربية**: تصميم متجاوب وواجهة عربية بالكامل
- **تصنيف متقدم**: تصنيف الآيات حسب التنزيلة والمجموعات

## الاستخدام

افتح ملف `index.html` في متصفح الويب:

```bash
# باستخدام Python
python -m http.server 8000

# أو ببساطة افتح index.html مباشرة في المتصفح
```

ثم افتح المتصفح على: `http://localhost:8000`

## الملفات

- `index.html`: الملف الرئيسي للتطبيق (يحتوي على HTML, CSS, JavaScript)

## المميزات التفصيلية

### العرض حسب التنزيلة
- عرض جميع التنزيلات المتاحة
- تصفح الآيات حسب التنزيلة المختارة
- تصنيف حسب المجموعات

### العرض حسب السورة
- عرض جميع سور القرآن
- تصفح الآيات حسب السورة
- عرض العناوين والمجموعات
